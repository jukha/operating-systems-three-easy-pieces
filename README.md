# OSTEP Chapter Index

A clean, single-page index of all chapter links for **Operating Systems: Three Easy Pieces (OSTEP)** by Remzi H. Arpaci-Dusseau and Andrea C. Arpaci-Dusseau.

🔗 **Live page:** https://jukha.github.io/operating-systems-three-easy-pieces/

## Why this exists

The book is free and official at **[ostep.org](https://ostep.org)**, but the chapter index on the official site is a plain HTML table that's hard to scan — small text, no visual grouping, and no hover states. All this page does is reorganize the same official links into a cleaner layout with color-coded sections (Intro, Virtualization, Concurrency, Persistence, Security) so it's easier to find and open a chapter.

This is **not a mirror or a copy** of the book. Every link points directly to the official PDFs hosted at `pages.cs.wisc.edu/~remzi/OSTEP/`. No content is hosted here.

## Official source

Just open `index.html` in any browser, or visit the GitHub Pages link above.
